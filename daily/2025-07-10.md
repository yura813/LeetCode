### 題目：Top K Frequent Element

- 初始想法：
  用 Map 存數字出現次數，要找次數最多的前 K 個，要排名不就要 sort 了嗎？
  要怎麼不 sort 也可以排序？

學到 **開個陣列用 index 的天然排序**

這是 **bucket sort 桶排序**，把值放在對應的 index 桶子裡，掃過陣列就等於排序結果
但只適用在能夠轉換成整數範圍的東西，有點統計轉換的感覺，例如出現次數

```
1-5 -> bucket[0]
6-10 -> bucket[1]
11-15 -> bucket[2]
```

但如果連桶子裡面都要排序的話，就會超過 O(n) 了
